REST API

API Reference

All REST endpoints grouped by service. JWT Bearer required on all endpoints unless marked Public. Identity is validated once at the YARP API Gateway and forwarded as headers (X-User-Id, X-User-Role, X-User-Email, X-Restaurant-Id) to downstream services.

Identity Service
MethodEndpointAccessDescription

POST/auth/forgot-passwordPublicSend 6-digit OTP to email — valid 10 min, single use
POST/auth/reset-passwordPublicVerify OTP and set new password

PATCH/users/meAuthenticatedUpdate name, phone, username, avatar

POST/users/staffAdmin · DirectorCreate waiter / chef / cashier / courier account for a restaurant

PATCH/users/{id}/statusAdmin · DirectorActivate or suspend user account
